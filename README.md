# ProjectX3D
ProjectX3D

Чтобы проект запускался из Rider нужно скопировать файл content.dat и папку Content
по пути ProjectX3D\bin\Debug\net8 (в зависимости от версии .net)
можно сразу запустить скомпилированный ProjectX3D.exe


Так же можно запустить прямо из папки с проектом командой dotnet run из консоли
